# CLI Reference

Every command writes its artifact under `output/<category>/` and prints the path. Shared
good-fraction options appear on most commands: `--max-size` (default 24), `--max-prime` (default
5), `--max-lcm` (default 24), `--ktet` (default 12).

## Tables

### `good-fractions`

```bash
melodroid table good-fractions [--max-size 24] [--max-prime 5]
```

Prints the good fractions for the given size/prime limits.

### `lcm-families`

```bash
melodroid table lcm-families [--max-size 24] [--max-prime 5] [--max-lcm 24]
```

Prints the LCM families, ascending by LCM, empty families omitted.

### `bin-overlaps`

```bash
melodroid table bin-overlaps [--max-size 24] [--max-prime 5]
```

For each adjacent pair of good fractions, the bin radius
$c = (\text{Upper} - \text{Lower}) / (\text{Upper} + \text{Lower})$ at which their JND bins first
touch. The caption reports the minimum $c$ — the binding constraint.

### `octave-sweep`

```bash
melodroid table octave-sweep --ratios 1.0 1.25 1.5 \
  [--sweep-step 0.001] [--bin-radius 1/161] [--only-full-matches] \
  [--only-centered-full-matches] [--max-size 24] [--max-prime 5]
```

Sweeps a reference ratio across $[1, 2)$, binning each input ratio against the good fractions;
flags full matches and centered full matches. `--ratios` is required (space-separated decimals on
$[1, 2)$).

### `ktet-cutoffs`

```bash
melodroid table ktet-cutoffs [--max-k 50] [--only-strictly-improving] \
  [--max-size 24] [--max-prime 5]
```

Lists the worst-case covering radius $c_k$ per $k$, its limiting fraction, and the nearest key.

### `key-sweep`

```bash
melodroid table key-sweep --ktet 12 (--keys 0 4 7 | --ratios 1.0 1.25 1.5) \
  [--bin-radius c_k] [--only-full-matches] [--max-size 24] [--max-prime 5]
```

Like the octave sweep but stepping through the $k$ keys of the tuning. At least one of `--keys` /
`--ratios` is required; both may be combined.

### `placement`

```bash
melodroid table placement --lcm 4 [4 ...] --at 0 \
  [--ktet 12] [--max-size 24] [--max-prime 5] [--max-lcm 24]
```

Prints the keys and fractions for one or more LCM placements at a shared anchor.

### `family-overlap`

```bash
melodroid table family-overlap --lcm-sweep 4 --lcm-ref 24 \
  [--ktet 12] [--max-size 24] [--max-prime 5] [--max-lcm 24]
```

Sweeps one LCM shape against a reference family; green rows sit wholly inside the reference.

### `key-supersets`

```bash
melodroid table key-supersets --keys 0 4 7 \
  [--ktet 12] [--max-size 24] [--max-prime 5] [--max-lcm 24]
```

Lists every placement whose keys contain the requested set, tightest fits first.

### `voicings`

```bash
melodroid table voicings (--lcm 24 | --keys 0 4 7) \
  [--ktet 12] [--max-size 24] [--max-prime 5] [--max-lcm 24]
```

Lowest-penalty voicings from each root. Exactly one of `--lcm` / `--keys`.

### `chord-melody`

```bash
melodroid table chord-melody --chord-keys 0 4 7 \
  [--ktet 12] [--max-size 24] [--max-prime 5] [--max-lcm 24]
```

Matrix of maximal-LCM placements containing the chord vs each key.

## Graphs

### `lcm-families`

```bash
melodroid graph lcm-families [--mode full|collapsed] \
  [--max-size 24] [--max-prime 5] [--max-lcm 24]
```

Writes a Mermaid relation graph to `output/graphs/`. `--mode collapsed` groups each isomorphism
class into a single node.

## Plots

### `lcm-families`

```bash
melodroid plot lcm-families --lcm 6 [--mode all|sum|constituents|difference] \
  [--subset-lcm K] [--difference-only] [--samples-per-period 200] \
  [--max-size 24] [--max-prime 5]
```

Writes a ScottPlot PNG of the family's sines and their superposition to `output/plots/`.
