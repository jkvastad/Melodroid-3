# Wave Pattern Plots

The LCM of an LCM family is its **wave pattern length** (WPL): the number of reference periods over
which the in-phase superposition of its fractions repeats. We can see this directly by plotting
each fraction $p/q$ as

$$
\sin\!\left(2\pi \cdot \tfrac{p}{q} \cdot t\right)
$$

(amplitude $1$, all in phase at $t = 0$) on a shared time axis from $0$ to $\mathrm{LCM}$,
together with their sum. Every component sine completes an integer number of cycles by
$t = \mathrm{LCM}$ and returns to zero, so the summed pattern closes at the endpoint — the wave
pattern visibly repeats with period LCM.

## Highlighting a sub-family

A literal-subset family can be highlighted inside the parent plot. Because a family's WPL equals
its LCM, whenever $K$ divides the parent LCM the sub-family's pattern tiles $\mathrm{LCM} / K$ times
across the parent window — e.g. LCM 4 fits twice into LCM 8, LCM 3 fits eight times into LCM 24.

```bash
melodroid plot lcm-families --lcm 24 --subset-lcm 3
```

Options: `--subset-lcm K` (the sub-family LCM to highlight), `--max-size` (default 24),
`--max-prime` (default 5). See the
[CLI reference](./reference.mdx#lcm-families-2) for the full command surface. $K$ must divide
the parent LCM, and the family at $K$ must exist under the current size / prime limits.

Blue dashed lines mark each multiple of $K$, and the sub-family's own superposition is drawn in
solid blue. Highlighting LCM 3 inside LCM 24 shows the eight tilings:

<WavePlot lcm={24} subsetLcm={3} />

## The residual (difference mode)

The **difference** mode isolates the residual — the parent's superposition minus the subset's.

```bash
melodroid plot lcm-families --lcm 24 --subset-lcm 4 --mode difference
```

Options: `--mode difference` (isolate the residual), `--subset-lcm K` (the subtracted sub-family),
`--difference-only` (drop the parent and subset curves, plot only the residual), `--max-size`
(default 24), `--max-prime` (default 5). See the
[CLI reference](./reference.mdx#lcm-families-2) for the full command surface.

By default it overlays three curves: the parent sum (black), the subset sum (blue), and the
difference (red), keeping the subset's dashed period markers for reference. Where the subset
captures the bulk of the pattern the residual stays small; where the parent adds genuinely new
structure it swings wide:
