# A Good Keyboard

Choosing the ratios for an instrument creates a **tuning system**. Because this is about
well-defined frequencies, we frame it as a keyboard for familiarity.

A key property is **modulation**: playing the same frequency ratios from different fundamentals
while producing the same harmonies. Formally, a keyboard should express every possible full
match from every key. An arbitrary keyboard with the good fractions available from one reference
point will not generally have them from another — but **equal-temperament** tuning fixes this,
since the ratio between adjacent keys is constant, making every fundamental equivalent.

Combining equal temperament with octave equivalence gives instruments whose frequencies have the
form $f_0 \cdot 2^{n/k}$, where $f_0$ is an arbitrary reference (e.g. standard pitch
$440\ \mathrm{Hz}$), $n$ is the signed key distance, and $k$ is the number of keys per octave.
Setting $k = 12$ gives the familiar 12-TET tuning.

## How many keys are needed?

List the requirements the theory places on the keys:

- **Modulation.** Every good fraction must be playable from any key as reference — relative to
  any key, each good fraction needs at least one key within its bin radius. Infinitely many keys
  solve this but maximise complexity; a single key with a half-octave radius also "works" but is
  far outside any JND.
- **Full expression — no unplayable virtual reference points.** A set of keys may produce a full
  match whose reference point is not itself a played key. For example the major chord
  $\{1, 5/4, 3/2\}$ has an LCM-8 full match at $4/3$. That virtual reference should be playable.
- **Minimal complexity — as few keys as possible.**

What does it mean to "play a good fraction using a key"? With a keyboard the ratio set is fixed
by the keys, and the reference point must also be a key. For a key set with a full match, the
reference point has a nearest key; if it is close enough we hear it as the reference. We take
"close enough" to be the same bin radius used for clustering: the sounded key, expressed as a
ratio, must bin to the target good fraction.

### Modulation

For a $k$-TET system and a reference key, each good fraction needs a key within its bin radius.
For a given $k$ there is a good fraction needing the largest radius to reach its nearest key —
the worst-case covering radius

$$
c_k = \frac{\lvert 2^{n/k} - g_k^* \rvert}{g_k^*},
$$

the multiplicative distance caused by the limiting fraction $g_k^*$.

```bash
melodroid table ktet-cutoffs
```

lists, for each $k$, the cutoff $c_k$, the limiting fraction $g_k$, and the nearest $k$-TET key
(index $n$ and ratio $2^{n/k}$). Green rows mark where $c_k$ strictly improves over every smaller
$k$. For the standard good fractions (max size 24, max prime 5):

| $k$ | $c_k$ | $c_k$ % | Limiting fraction | Nearest $n$ | $2^{n/k}$ |
| ---: | ---: | ---: | :--- | ---: | ---: |
| 11 | 0.029332 | 2.9332 % | 5/4 | 4 | 1.286665 |
| 12 | 0.010216 | 1.0216 % | 10/9 | 2 | 1.122462 |
| 19 | 0.008460 | 0.8460 % | 16/15 | 2 | 1.075691 |
| 31 | 0.006458 | 0.6458 % | 10/9 | 5 | 1.118287 |
| 34 | 0.004547 | 0.4547 % | 9/8 | 6 | 1.130116 |

Since the JND is almost $1\%$, $k = 11$ gives about $3\times$ JND but $k = 12$ suddenly drops to
JND. It only improves slightly (~20% smaller) at $k = 19$, then again at $31$ and $34$ — around
the threshold for unique good-fraction bin radii. So $k = 12$, with about $63\%$ of the keys of
$k = 19$, is a strong compromise. Remarkably, we have recovered the modern popular tuning system
from first principles of biology and physics.

### Full expression

Any key set with a full match must be able to play a key that bins to its reference point. Since
the modulation solution already lets every key act as a reference for every good fraction,
sufficiently low-radius $k$-TETs satisfy this automatically.

### Minimal complexity

We want the fewest keys meeting the other requirements. The viable $k$-TETs are $12, 19, 31, 34$,
favouring **12-TET** — with two caveats:

- the bin radius may be too large, forcing 19 or even 34 (unique binnings) keys;
- the solution may be *too* perfect, allowing too little dissonance (recall the chili). Effects
  like string bending introduce sliding dissonance that listeners enjoy. 12-TET still allows
  some: none of the 13 standard good fractions map to key 6 (the tritone, the "Devil's
  interval"), and keys 2 (10/9, 9/8) and 10 (16/9, 9/5) have ambiguous mappings.
